FAQ

  • Ваша задача – написать код ИИ для своих юнитов
  • Есть три класса юнитов: маги, стрелки и пехотинцы
  • Код – чистый JS, исполняется последовательно для каждого юнита
  • Код можно запускать для своих юнитов до отправки
  • Очередность хода зависит от скорости юнита, быстрые ходят первыми
  • После нажатия "Готово" обоими игроками, оба кода будут запущены на общем игровом поле

API

goTo(x: number, y: number)
Переход на клетку x, y
relativeGoTo(x: number, y: number)
Переход на клетку x, y относительно позиции юнита
goToEnemyAndHit(id: string)
Переход в сторону противника по id и попытка атаки, если это возможно
пехотинец атакует только если может дотянуться до противника
shoot(id: string)
Стрелковая атака противника по id
применимо только к стрелкам
spell(id: string)
Магическая атака противника по id
применимо только к магам
say(text: string)
Сказать что-нибудь text
attackRandom()
Берсерк! Случайный выбор юнита на карте и атака!
пехотинец атакует только если может дотянуться до противника
isShooter(): boolean
Проверка, является ли юнит стрелком
isMagician(): boolean
Проверка, является ли юнит магом
isInfantry(): boolean
Проверка, является ли юнит пехотинцем
isAlive(): boolean
Проверка, является ли юнит живым
getHealth(): number
Возвращает здровье юнита от 0 до 100
getID(): string
Возвращает ID юнита
getX(): number
Возвращает X позицию юнита
getY(): number
Возвращает Y позицию юнита
is(id: string): boolean
Возвращает true если ID юнита равен id
